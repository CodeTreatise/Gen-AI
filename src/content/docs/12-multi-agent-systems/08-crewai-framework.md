# 12.8 CrewAI Framework

## What is CrewAI
- Role-based agent framework
- Crew and task concepts
- Process-driven execution
- Real-world team simulation
- Easy-to-use abstraction

## Installation and Setup
- pip install crewai
- crewai[tools] for tools
- Environment configuration
- LLM provider setup

## Agent Definition
- Role specification
- Goal definition
- Backstory for context
- LLM configuration
- Verbose mode
- Tool assignment

## Task Definition
- Task description
- Expected output
- Agent assignment
- Context from other tasks
- Output files
- Async execution

## Crew Composition
- Assembling agents into crew
- Task assignment
- Process type selection
- Verbose logging
- Memory settings

## Process Types
- Sequential process
- Hierarchical process
- Manager agent in hierarchical
- Custom processes
- Process selection criteria
