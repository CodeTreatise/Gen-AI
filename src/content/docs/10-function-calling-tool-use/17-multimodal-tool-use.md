# Multimodal Tool Use

- Vision with function calling
  - Images as input with tool definitions
  - Analyze image then call function
  - Screenshot analysis
  - Document understanding
- Multimodal function responses
  - Functions returning images (Gemini 3)
  - Documents in responses
  - `inlineData` with MIME types
  - `displayName` for references
- Supported MIME types
  - Images: image/png, image/jpeg, image/webp
  - Documents: application/pdf, text/plain
  - Provider-specific support
  - Size limitations
- Computer use capabilities
  - Browser automation
  - UI interaction
  - Screenshot analysis
  - Action execution
  - Safety considerations
