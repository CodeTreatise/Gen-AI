# Fundamentals of Effective Prompts

- Clarity and specificity
  - Avoiding ambiguous language
  - Precise terminology
  - Concrete examples in prompts
  - Scope definition
  - Expected output length hints
- Role and context setting
  - "You are a..." patterns
  - Domain expertise assignment
  - Audience definition
  - Background information
- Instruction ordering
  - Most important instructions first
  - Logical flow of requirements
  - Grouping related instructions
  - Numbered vs. prose instructions
- Explicit vs. implicit instructions
  - When to be explicit
  - What models infer well
  - Cultural and context assumptions
  - Testing implicit understanding
- Positive vs. negative framing
  - "Do X" vs "Don't do Y"
  - When negation is necessary
  - Combining positive and negative
  - Avoiding double negatives
- Conciseness vs. detail balance
  - When more detail helps
  - Diminishing returns on length
  - Key information identification
  - Iterative refinement
