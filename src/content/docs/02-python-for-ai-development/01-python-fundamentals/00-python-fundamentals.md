# Python Fundamentals

## Overview

Python's clean syntax and readability make it the language of choice for AI and data science. Understanding Python fundamentals is essential before diving into AI libraries and frameworks.

This lesson covers core Python concepts that every AI developer needs to know.

---

| Lesson | Topic | Key Concepts |

## Why Python for AI?

| Advantage | Description |
|-----------|-------------|
| **Readability** | Clean syntax, easy to learn and maintain |
| **Rich ecosystem** | NumPy, Pandas, TensorFlow, PyTorch, LangChain |
| **Community** | Largest AI/ML community, abundant resources |
| **Interoperability** | Easy integration with C/C++, REST APIs |
| **Rapid prototyping** | Quick experimentation and iteration |

## Python Version

This course uses **Python 3.10+** for features like:
- Match statements (pattern matching)
- Improved error messages
- Type union syntax (`int | str`)
- Parenthesized context managers

```bash
# Check your Python version
python --version
# Python 3.12.x recommended
```
