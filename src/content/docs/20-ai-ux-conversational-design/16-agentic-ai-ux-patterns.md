# Agentic AI UX Patterns (NEW 2025-2026)

- Multi-step task visualization
  - Task breakdown display
  - Step-by-step progress indicators
  - Current action highlighting
  - Estimated completion per step
- Tool use transparency
  - Tool call visualization
  - What tool is being used
  - Tool execution status
  - Tool output display
- Action scaling UX
  - Extended agent sessions
  - Computer use visualization
  - Screenshot/action history
  - Iterative improvement display
- Autonomous coding agent UX
  - GitHub Copilot coding agent patterns
  - Issue-to-PR automation visualization
  - Background processing with notifications
  - Human review integration points
- Agent failure and recovery
  - Multi-step failure handling
  - Partial completion states
  - Rollback options
  - Human takeover triggers
- Agent mode interfaces
  - IDE agent mode (VS Code, JetBrains)
  - Terminal agent workflows
  - CLI AI assistants
  - MCP (Model Context Protocol) integration
